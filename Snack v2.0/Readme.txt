1. Game Description
-------------------

This is a simple api snack game. This game uses rectangle, ellipses, lines to draw
the game elements such as: Game Map, Snack nodes, Food.

That game starts with drawing the snack in the middle of the Game map and reveal 
every distinct time a food, when you eat the food the snack become larger and your
score increase. the game terminates when you hit your self.

2. Controls
-----------

Key Controls:

    Esc            - Exit
	Up             - W
	Down           - S
	Right          - D
	Left           - A
